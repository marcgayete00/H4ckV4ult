IMAP (143/TCP, 993/TCP) POP3 (110/TCP, 995/TCP)

IMAP allows online management of emails directly on the server and supports folder structures.POP3, on the other hand, does not have the same functionality as IMAP, and it only provides listing, retrieving, and deleting emails as functions at the email server.

Enumeration

    Performing basic scan in order to detect data

    nmap 10.129.14.128 -sV -p110,143,993,995 -sC

    Connect to server

      If we would like to connect to webserver we should use the following command (Recomended since this one allows to interact with the mail server)

      openssl s_client -connect 10.129.242.130:imaps  #IMAP
      openssl s_client -connect 10.129.14.128:pop3s    #POP3

      Also in case we have valid credentials we could perform: (You are only able to list the different mailboxes)

      curl -k 'imaps://10.129.14.128' --user user:p4ssw0rd

    Usage

      Once we have performed a connection with the previously mentioned commands we could perform:

      1 LOGIN username password
1 LIST "" *  #Lists all INBOXES
1 SELECT INBOX  
1 FETCH ID all  #Show message headers (From, to, concept)
1 FETCH 1 BODY[TEXT] #Show text of the message
