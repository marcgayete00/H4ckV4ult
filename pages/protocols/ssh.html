SSH (TCP/22)

SSH (Secure Shell) is a network protocol that establishes encrypted connections between computers for secure remote access

Interesting files

    In some exploitations you may find a private key file which is super important. Using it we can stablish an SSH connection without using any password

    Windows typical location

      C:\Users\$USERNAME\.ssh

      File is normally called

      id_rsa

    Linux typical location

      /etc/ssh/

      File is normally called

      ssh_host_rsa_key
