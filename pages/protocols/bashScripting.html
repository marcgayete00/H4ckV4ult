Bash scripts are files containing commands that you run in the terminal. They automate tasks and make your work more efficient.

    Conditionals

      There are several conditionals to perform comparations here comes the list:

        Logical operators

      • ! logical negotation NOT
      • && logical AND
      • || locagical OR

        • Integer operators

      • -eq is equal to
      • -ne is not equal to
      • -gt is greater than
      • -ge is greater than or equal
      • -lt is lower than
      • -le is lower than or equal
      • != is different than

        • String operators

      • == is equal to
      • != is not equal to
      • < is less than in ASCII alphabetical order
      • > is greater than in ASCII alphabetical order
      • -z if the string is empty (null)
      • -n if the string is not null

        • File operators

      • -e if the file exist
      • -f tests if it is a file
      • -d tests if it is a directory
      • -L tests if it is if a symbolic link
      • -N checks if the file was modified after it was last read
      • -O if the current user owns the file
      • -G if the file’s group id matches the current user’s
      • -s tests if the file has a size greater than 0
      • -r tests if the file has read permission
      • -w tests if the file has write permission
      • -x tests if the file has execute permission

    Looping methods

      A loop consists in a operation that is used to perform one o serveral actions repeatedly. Some examples are

      For loop

        This one is used when you need to perform a static number of operations. For example 40

        for counter in {1..40}
do
    #Perform operations here
done

        Another example of looping with for is, iteration between arguments

        for arg in "$@"
do
    echo "Argumento: $arg"
done

      While loop

        This one is used then you want to perform the action repeatedly after a trigger shows up

        counter=1
while [ $counter -le 5 ]
do
  echo "Welcome $counter times"
  counter=$(( $counter + 1 ))
done

      It is important to mention that in order to increase the value of the variable inside the loop we should perform someting like:

      $variable=$(( $counter + 1 )) ||  OR  || var=$(echo $var | base64)

    Exit methods

      When attempting to exit in a part of a script we can use the special word exit exit comes with a number that indicates how the script exited. Here you can find the diferent exit numbers

      • Value 0 - represents successful execution
      • Value 1 - represents a result of a failure
