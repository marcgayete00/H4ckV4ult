    PWM (Password Management Servlets)

      Password Management Servlets (PWM) is an open-source Java-based software designed to provide self-service password and account management in enterprise environments with LDAP authentication.

      In case you found one of this Password Management Servlets you could perform two things

      • In some cases there will be an option to test the LDAP connection, in this case you could attempt to modify the target URL to make it point to your attacker machine and then capture the connection with responder. If you are lucky you could be able to capture clear text credentials for the svc_ldap account
      • Once you have valid credentials to managed the application attempt to identify the .xml configuration files where the password for svc_ldap user is found. In case you found it you could attempt to crack it
