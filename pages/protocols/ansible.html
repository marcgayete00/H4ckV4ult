Password cracking

    In case you found password hashes of ansible pltaform you could be able to crack them. But first follow the format to crack it successfully.

    $ANSIBLE_VAULT;1.1;AES256
63303831303534303266356462373731393561313363313038376166336536666232626461653630343733303536623561343737373

    Now apply john to converit to a compatible format in order to crack the vault password

    ansible2john ldap_admin_password > hash3.txt #Convert it
john hash3.txt --wordlist=/usr/share/wordlists/rockyou.txt #Crack it

    Then use the following command and provide the cracked password found previously

    ansible-vault decrypt ldap_admin_password #File with the ansible password hash

    Have in mind that the clear text password will be replaced inside the file
