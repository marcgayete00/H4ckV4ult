Splunk (TCP/8000, TCP/8089)

Splunk is a log analytics tool used to gather, analyze and visualize data.

Enumeration

    Default password admin:changeme Latest versions of splunk set up password during installation

Exploitation

    An easy way to gain remote code execution in a splunk server is trough creating a custom application to run python, batch or powershell scripts

    To do so we can use the following github repo

    Inside the repo will found a folder which will contain to folders inside /bincontains the scripts for Linux (python) and Windows (Batch and Powershell). The /default contains the configuration file which will tell that the application runs every 10 seconds

    Based on the system we are attacking will have to modify the IP adress of the files located at the /bin folder

    Once all is configured archive the folder using and setup a netcat connection at the port specifyed

    tar -cvzf updater.tar.gz splunk_shell/

    Then upload the app by going from the main page to Splunk apps-> Apps (Top part) -> Manage apps-> Install app from file Browse your malicious app and install

    Finally you'll recive your shell
