Nagios (TCP/5666, TCP/5667)

Nagios is a leading open-source IT infrastructure monitoring system that tracks networks, servers, applications, and services

Enumeration

    Version identification

      The version of nagios can be found once logged in. Even though performing some directory fuzzing you'll be able to find it

Exploitation

    CVE-2021-25296, CVE-2021-25297, and CVE-2021-25298

      Nagios 5.7.5 is affected by different CVEs which lead to remote code execution over the server. Run the following plugin in metasploit to exploit it

      exploit/linux/http/nagios_xi_configwizards_authenticated_rce

      In this case you are able to change the exploit method by changing the affected CVE in the module. Try the three of them in case any of them does not work
