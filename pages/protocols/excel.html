Excel

A software program created by Microsoft that uses spreadsheets to organize numbers and data with formulas and functions

Exploitation

    Ofuscated data

    In some cases you will find that excel seems to be cyphered/offuscated so you can't see data clearly

    Rename file to .zip and navigate to xl folder

    xl

    Inside this folder you will find clear text data, Shared Strings, Worksheets

    Password Protected

    Can't access your excel? Password protected?

    Rename file to .zip and navigate to xl folder then access Worksheets

    Select the worksheet you can to remove password and open it with a CodeEditor

    Serch for "Protect" string and youll find the following XML tag

    <sheetProtection scenarios="1" objects="1" spinCount="10000" saltValue="VALUE" hashValue="VALUE" algorithmName="SHA-512">

    Remove that line, save file, rename file agant to .xlsx and open it
