Legacy Systems

This page pretends to document several vulnerabilities that may occur in legacy systems such as Windows XP, 2003, old unix distributions...

Windows XP, 2000 2003, Vista, 2008

    MS08-067

    Allows an attacker to send SMB especially crafted packages in order to remote code execution. Used by conficker worm

    Metasploit module

    use exploit/windows/smb/ms08_067_netapi

    MS17-010 (Eternalblue)

    Allows an attacker to send SMB packages and gain RCE in the victim

    This can only be used in 64 bit architecture

    Metasploit module

    windows/smb/ms17_010_eternalblue
