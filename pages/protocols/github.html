Github

A web-based platform that uses the Git version control system to help developers store, manage, and collaborate on their code

Usage

    Here there are a few commands to use git in terminal in order to perform some actions

    Clone repo

      git clone URL

      You can always clone a ssh repo 

      git clone ssh://bandit27-git@bandit.labs.overthewire.org:2220/home/bandit27-git/repo

    Check log

      Show all changes performed in a repo

      git log

      In case we want to check changes for a specific commit we could do

      git show 710c14a2e43cfd97041924403e00efb00b3a956e

    Show branches

      Show all branches from a repo

      git branch -a

      In case we want to change the branch were we are (Where dev is branchname)

      git checkout dev

    Show tags

      Show all tas of a repo

      git tag

      In case we want to check content of a tag (Where secret is the name of the tag)

      git show secret

    Perform a push

      How to perform a push

      Once you have the file we want to push we will need to add it to the repo

      git add -f key.txt

      Perform a commit with any kind of text we want

      git commit -m "Add key.txt with secret content"

      Just perform the push

      git push origin master

    Enumeration

      Last commit

        In case you found a /.git o similar in a web you could be able to acquire all files and undo last commit

        wget -r http://website/git

        Then using git cola select the folder where you extracted all the content and look for intersting files such as settings.php

        In case you would like to undo last commit just click in "Commit > Undo Last Commit"
