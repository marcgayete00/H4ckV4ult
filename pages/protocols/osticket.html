Ticketing system

Information gathering

    Once you get access to the ticketing system it is highly recommended to search for closed tickets which could contain credentials. To do so filter using the word password or similar

Exfiltration of information

    In some cases the main website of osticket allows you create and view information about tickets. In that case I suggest you to create an account and attempt to generate a ticket. Once done it check what osticket says since it can give you useful information. Possible text

    You may check status of ticket by navigating to the check status page using id : 111111 if you want to add more information to the ticket just email 11111@delivery.htb

    This means that osticket created an account with mail 11111@delivery.htb and that any thing we send to that mail account will be listed inside the ticket history since the information will be updated

    Knowing that we could be able to send a password reset using the mail 111111@delivery from another platform in order to leak the information in the ticket history
