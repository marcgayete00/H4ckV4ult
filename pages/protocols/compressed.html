Compressed Files

Compressed files are files that have been reduced in size to save storage space or make transfer faster. Compression works by removing redundancies in the data, and it can be either lossless (no data is lost, e.g., ZIP) or lossy (some data is lost to reduce size, e.g., MP3, JPEG).

Common extensions ZIP RAR 7Z TAR GZ

Usage

    ZIP

    unzip file.zi

    RAR

    unrar x file.rar

    7Z

    7z x file.7z

    TAR

    tar -xvzf archivo.tar.gz

    GZ

     gunzip archivo.gz

Exploitation

    AES encrypted file

    In case you find a file like:

    file.zip.aes

    You can decode it using

    pyAesCrypt -d file.zip.aes
