Mattermost (TCP/80) (TCP/443)

Mattermost is an open-source, self-hostable secure collaboration and messaging platform designed for technical, operational, and enterprise teams

Usage

    This app is very similar to teams or other apps but with a tweak of more technical features

    Things to check

      As any other messaging app it is very important to check message history, but in this case we must check the integrations section which in some cases maay lead to command execution or other

Password cracking

    Mattermost stores passwords in a database Mysql or MariaDB those ones are stored inside the table users. In order to crack them we should use the following hashcat module

    hashcat -m 3200 hash -r /usr/share/wordlist/rockyou.txt
