A Network File System (NFS) is a technology that lets users access files on a remote server as if they were stored locally on their own computer

Enumeration

    List contents

    showmount -e 10.129.14.128

    Once you know what folder would you like to mount we must create a folder and perform the mounting

    mkdir target-NFS
sudo mount -t nfs 10.129.14.128:/ ./target-NFS/ -o nolock
cd target-NFS
