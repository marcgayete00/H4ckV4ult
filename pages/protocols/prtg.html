PRTG Network Monitor (TCP/80, TCP/443)

PRTG Network Monitor is agentless network monitor software. It can be used to monitor bandwidth usage, uptime and collect statistics from various hosts, including routers, switches, servers, and more.

Enumeration

    Default admin username is prtgadmin, password may bary so we should bruteforce eventhough you may try Password123

Exploitation

    CVE-2018-9276 versions before 18.2.39

      This CVE can be exploited due a lack of sanitization when passing a parameter during the creation of a notification. Here we explain how to exploit it

      First go to Setup->Account settings->Notifications. Once inside click on Add new notification

      Give a notification name and scroll down to Execute program, activate it and introduce the following parameters

      Program file: Demo exe notification - outfile.ps1
Parameter: test.txt;net user prtgadm1 Pwn3d_by_PRTG! /add;net localgroup administrators prtgadm1 /add

      In the parameter is where the injection comes. In this case we are creating a new local administrator but we could introduce a reverse shell for example

      Finally from the notification panel select the malicious notifications and click on Send notification test which is at the left part with a bell icon
