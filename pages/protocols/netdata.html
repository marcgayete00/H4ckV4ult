Exploitation

    CVE-2024-32019

      The following versions are affected `>= v1.45.0`, `= v1.44.0-60`, `< v1.45.0-169` In affected versions the `ndsudo` tool shipped with affected versions of the Netdata Agent allows an attacker to run arbitrary programs with root permissions since the executable nsudo has with the SUID bit set

      The process to exploit it is the following:

      • Compile a malicious script in C and perform a system command execution, reverse shell o what ever you want to do with root permissions
      • Name it nvme and Transfer the executable into the victim machine
      • Provide +x exectuion permissions
      • Update the path adding the path where the malicious executable is
        • PATH=/home/oliver:$PATH
      • Run the ndsudo binary which automatically will make a call to our malicious executable called nvme and execute our crafted code
        • /opt/netdata/usr/libexec/netdata/plugins.d/ndsudo nvme-list
