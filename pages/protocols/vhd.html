VHD (Virtual Hard Disk)

WinRM stands for Virtual Hard Disk. It is a file format that contains an entire hard drive, primarily used to store the operating system and data of a virtual machine.

Usage

    You may find this files alone or by couple. Normally one file belongs to the partition system (which does not contain valuable information about anything)

    And anotherone which contain the whole file system itself which means we can find anything inside the virtual disk

    To navigate inside this files we'll need to use the following commands

    First of all we will need to install qemu-utils

    sudo apt update
sudo apt install qemu-utils

    Then we'll load the nbd module into the kernel in order to make a virtual file behave as a local block

    sudo modprobe nbd max_part=8

    Then connect the VHD file to our filesystem

    sudo qemu-nbd --connect=/dev/nbd0 file.vhd

    A drive will appear into the desktop, open it an navigate...
