R-Services (TCP/512 ,TCP/513 ,TCP/514)

R-Services are a suite of services hosted to enable remote access or issue commands between Unix hosts over TCP/IP.The R-services were the de facto standard for remote access between Unix operating systems until they were replaced by the Secure Shell (SSH)

The suite consists of the following commands

  • rcp (remote copy)
  • rexec (remote execution)
  • rlogin (remote login)
  • rsh (remote shell)
  • rstat
  • ruptime
  • rwho (remote who)s

Exploitation

    The most important file to look for is called hosts.equiv which is recognized as the global configuration regarding all users on a system, whereas .rhosts provides a per-user configuration.

    Logging in Using Rlogin

      In order to find that user it is necesary to check host.equiv file since it contains this kind of names

      rlogin 10.0.17.2 -l htb-student

    Listing authenticated Users

      Once in you are able to list users using rwho command

      rwho

      Another way of listing users is by using rusers command

      rusers -al 10.0.17.5
