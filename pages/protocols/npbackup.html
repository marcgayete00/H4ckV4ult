Npbackup

Another backup tool based on restic. This is designed for sys admins or for final users

Other similar tools backrest

Exploitation

    Visualizase content of files

    In case you have permisions to create backups you'll be able to restore content and visualice content of files which sometimes may be useful

    Create backup. Remember to have the config file aiming to the folder you would like to backup.

    sudo npbackup-cli -c file.conf -b

    Check if backup was correctly created. With that command you can also check the which files were backed up 

    sudo npbackup-cli -c file.conf --ls

    Finaly show content of specific file

    sudo npbackup-cli -c file.conf --dump /root/root.txt
