NTLM

NTLM stands for NT LAN Manager, and it's a suite of Microsoft security protocols that provides authentication, integrity, and confidentiality for users and computers in Windows-based networks.

It is used:

- In Legacy systems

- When there is not DC so authentication is made locally

- When Kerberos configurations failed

Check if NTLM is active

smbclient -L $victim -U Administrator

Exploitation

NTLM works sending challenge/response to the server. In this process a hash with the password of the user. To do so we need to create a resource that can be accesed from de victim. To do so we created SMB server using responder 

sudo responder -I tun0

and accesed a resource from the server we just setup using a LFI in the victim to let responder intercept de hash
