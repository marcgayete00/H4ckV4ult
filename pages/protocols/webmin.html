Webmin (TCP/10000)

Webmin is a free, open-source, web-based control panel for Linux/Unix system administration. It provides a graphical interface (GUI) to manage server tasks, such as user accounts, Apache/Nginx, MySQL/PostgreSQL, DNS, and file systems, eliminating the need to manually edit configuration files or use the command line.
