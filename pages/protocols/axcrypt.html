AxCrypt

AxCrypt is a popular, user-friendly file encryption and cloud security software

Password cracking

    In case you found a file called something.zip.axx or with the axx extension you may be in front of a AxCrypt encrypted file. But don't worry if the encryption key is weak enough we could be able to crack it using john

    First convert the file in a compatible format for john to read it

    axcrypt2john yourfile.axx

    Now attempt to crack it

    john axcrypt_hash --wordlist=/usr/share/wordlist/rockyou.txt
    Once you found the key i highly recommend you to download the axcrypt client in a windows environment and open the file from there. Don't mess up with other softwares or tools which will not work
