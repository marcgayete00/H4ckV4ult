Macros

Almost all softwares from office allow users to use macros which are a set of instructions that can be embeded in documents to automate tasks. From our attacker side we could be able to operate with the operating system layer in order to execute commands or stablish reverse shell connections. Here we'll show you how create macros, but first we must learn when those macros are going to be executed

When macros are being executed

    If we successfully manage to deliver the Office document to our target via mail or download link have in mind that the document will be tagged with the Mark of the Web (MOTW) which is basically a tag that tells office that document you are about to open was downloaded from internet which means it could contain malware. When office detects that MOTW it opens the document with the protected view (yellow banner). In this state macros are disabled but when the user clicks in "enable editing" all macros are executed

How to create macros

    First we need to create a compatible document. Create a new word document and choose the following formats

    • Word 97-2003 Document (.doc) - This kind of document contains embedded macros
    • Word 97-2003 Document (.docm) - This kind of document contains embedded macros
    • Word Document (.docx) - This kind of document does not save macros meaning the macro in not persitent

    In order to create a new macro we must go to View and select Macros. Now create a new macro and setup the name you want. Now double click in macro name or click in edit
