Restic

Restic is a modern, fast, secure, and efficient open-source command-line backup program that supports Linux, macOS, Windows, and BSD

Backup gathering

    Windows

    In case restic is installed in the target system. (Check it typing restic in the terminal) we can try to recover backups performed

    First we need to identify the restic folder. Look for a folder called restic or similar where there are the following folders data, index, key, locks, snapshots

    Restic works with a password which we can setup as an environment variable. Once you found it use the follwoing command

    $env:RESTIC_PASSWORD = 'Superbackup!'

    Now list the snapshots.

    restic -r E:\restic\ snapshots

    Indentify the snapshot you would like to restore. Take a look at the folders which are being backed up. Restore it the following way

    restic -r E:\restic\ restore b2f5caa0 --target C:\Users\jeff\Desktop\new\
