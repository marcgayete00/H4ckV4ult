Backdrop (TCP/80)

Backdrop CMS is a free and Open Source Content Management System that helps you build modern, comprehensive websites on a reasonable budget.

Enumeration

    Directory listing

      Backdrop may not be configured properly in that case an attacker could perform a directory listing. Interesting directories can be found

      /core
/files
/.git

      An interesting file to checkout is update.settings.json it may contain users

Exploitation

    Remote code execution

      The version 1.27.1 it is affected by an authenticated remote code execution Can be found here

    Privilege Escalation

      Backdrop internally uses bee which is a command line utility for Backdrop CMS. In case we can run the binary using sudo permisions we can execute a shell with root

      sudo bee --rot=/var/www/html eval 'system("/bin/bash");'
