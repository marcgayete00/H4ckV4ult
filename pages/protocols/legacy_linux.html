Legacy Systems Unix

This page pretends to document several vulnerabilities that may occur in legacy systems such as Debian 6, Ubuntu 10.04, Fedora 14, SUSE 11

Shellshock (CGI)

    A Common Gateway Interface (CGI) is used to help a web server render dynamic pages and create a customized response for the user making a request via a web application. This is found in form of script normally located in Linux systems in /cgi-bin folder

    Extensions type could be. Windows: .cgi .bat. cmd .exe .pl .py .cgi | Linux: .cgi .pl .py .sh .cgi

    In case we found any script located in the /cgi-bin directory we should test the following to check if the system is vulnerable

    curl -H 'User-Agent: () { :; }; echo ; echo ; /bin/whoami' bash -s :'' http://10.129.205.27/cgi-bin/access.cgi

    Remember that shellshock is only exploitable if the server uses CGI, that CGI executes bash, bash is old and the server pass the http headers to environment variables
