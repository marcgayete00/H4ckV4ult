Bandit Challange (Overthewire) - Walkthrough

Level 0

    Connecting to server

    ssh bandit0@bandit.labs.overthewire.org -p 2220

Level 1

    Just perform a cat in the readme file

Level 2

    To read the file with the strange name 

    cat /home/bandit1/-

Level 3

    To read the file

    cat /home/bandit2/--spaces\ in\ this\ filename--

Level 4

    List hidden files, access inhere directory and list its content

Level 5

    List content of files jsut like this and so on

    cat /home/bandit4/inhere/-file01

Level 6

    Use the command find with the following parameters to filter by size

    find / -type f -size 1033c 2>/dev/null

    Then simply cat the file

    cat /home/bandit5/inhere/maybehere07/.file2

Level 7

    Again use the following command to filter by size 

    find / -type f -size 33c 2>/dev/null | grep bandit7

Level 8

    Use grep to filter the text

    cat data.txt | grep millionth

Level 9

    Sort file and apply uniq -u

    sort data.txt | uniq -u

Level 10

    Use grep to filter the file, since the file is binary use the flag -a

    grep -a “=” data.txt

Level 11

    Simple decode it from base64

Level 12

    Simply decode it from ROT13

Level 13

    The file is compressed multiple times...

    First base on the hexdump provided we create a file named data

    xxd -r /home/bandit12/data.txt data

    using file we discovered tar is a gzip file so

    mv data data.gz | gunzip data.gz

    use file again and we discovered is bzip2 file

    mv data data.bz2 | bunzip2 data.bz2

    Using file we discovered it is a tar file

    mv data data.tar | tar -xf data.tar

    Using file we discovered is tar again

    tar -xf data5.bin

    Using file we discovered is a bzip2

    mv data6.bin data.bz2 | bunzip2 data.bz2

    Using file we discovered it is a tar file 

    tar -xf data

    Using file we discovered it is a gzip file

    mv data8.bin data8.gz | gunzip data8.gz

    Finally using cat the password seems to be...

Level 14

    Using the flag -i we can use the provided key to login into bandit14

    Remember to change the the permisions chmod 700 sshkey.private in order to use the key

    ssh -i sshkey.private bandit14@bandit.labs.overthewire.org -p 2220

    Then performing a cat into /etc/bandit_pass/bandit14

Level 15

    Just perform a telnet connection using telnet and provide the previous password 

    telnet localhost 30000

Level 16

    Performing a connection with SSL and providing the previous password we complete the level

    openssl s_client -connect localhost:30001

Level 17

    Scan network with the provided range

    nmap -p31000-32000 localhost

    identify ports,enter and provide the password

    openssl s_client --connect localhost:31790

    Then they will provide you with an SSH key

Level 18

    Just use diff to compare both files

    diff passwords.old passwords.new

Level 19

    Since they modified the .bashrc to logyou out we can use SSH command to execute commands before the connection

    ssh bandit18@bandit.labs.overthewire.org -p 2220 "cat readme"

Level 20

    Using the binary file we can execute commands as another use, so we show the password located at /etc/bandit_pass/bandit20

    ./bandit20-do cat /etc/bandit_pass/bandit20

    Then performing a cat into /etc/bandit_pass/bandit14

Level 21

    Since the binary performs a connection to the port we want and compares the string provided se used nc

    echo "0qXahG8ZjOVMN9Ghs7iOWsCfZyXOUbYO" | nc -l -p 12345
    ./suconnect 12345

Level 22

    Looking at cronjob_bandit22 located in /etc/cron.d

    we see a binary called /usr/bin/cronjob_bandit22.sh

    Listing its contents, we found that it is making a cat over the file which contains the password from the next level and saving at 

    /tmp/t7O6lds9S0RqQh9aMcz6ShpAoZKF7fgv

Level 23

    Analyzing the script found we execute the command to get the md5sum 

    echo I am user bandit23 | md5sum | cut -d ' ' -f 1

    Result: 8ca319486bfbbc3663ea0fbe81326349

    Using the hash we know the name of the password file

    cat /tmp/8ca319486bfbbc3663ea0fbe81326349

Level 24

    Creation of script 

    echo -e '#!/bin/bash\ncat /etc/bandit_pass/bandit24 | nc localhost 4444' > /var/spool/bandit24/foo/sendpass.sh && chmod +x /var/spool/bandit24/foo/sendpass.sh

    Use nc to recieve the password

    nc -lvnp 4444
